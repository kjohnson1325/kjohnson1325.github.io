Cyclistic Case Study

This case study is for a fictitious bikeshare company called "Cyclistic". The team at cyclistic is gearing up for a marketing campaign targeting casual members and trying to convert them to annual members, which the Finance Team has shown is signicantly more profitable for the company. The Data Analysis Team is tasked with answering the following question - "How do annual members and casual riders behave differently?" This information will be used by the Marketing Team which will then design and execute a marketing campaign to target casual riders.

Since Cyclistic is not a real company, it has no real data. Luckily, there is data that is readily available from the Divvy bikeshare program in Chicago, IL. The raw data is in .csv file format and is stored in separate files for each month. The Analysis Team retreived data for one full year (July 2023 through June 2024). The raw data was cleaned, transformed, analyzed, verified, and visualized through a combination of SQL (BigQuery), Excel, and PowerPoint. The project files are shown below:

Covid-19 Data Exploration

This project is an initial exploration into COVID-19 data from January 2020 through April 2021. This data was retrieved from the World Health Organization (WHO), reporting the daily case counts and deaths around the world. The dataset was created by ourworldindata.org and can be found here.

The goal of this project was to complete an initial exploration of the data, sharing key insights using a data visualization. The data was imported, cleaned, transformed, and analyzed using SQL (MYSQL). Then some key tables were imported into Tableau and were used to create a dashboard. The project files are shown below:
